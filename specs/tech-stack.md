# 🛠️ Tech Stack

## Core Frameworks
- **React 19**: Leveraging the latest features for optimal performance and concurrency.
- **Vite**: Used for near-instant HMR and lightning-fast builds.

## State Management
- **Redux Toolkit**: Scalable state management with predictable updates.
- **Redux-Persist / LocalStorage**: Ensuring user settings and journal data survive page refreshes.

## Styling & Design
- **Tailwind CSS v4**: Modern, utility-first CSS for a bespoke design system.
- **Design Language**: Glassmorphism (translucency, subtle gradients, and high contrast).

## Reliability & Quality
- **TypeScript**: Strict typing to prevent runtime errors in financial math.
- **Vitest**: Fast, modern testing framework for unit and integration tests.
- **i18next**: Native support for multi-language (EN, VN).

## Infrastructure
- **Vercel**: Optimized deployment for frontend-heavy applications.
